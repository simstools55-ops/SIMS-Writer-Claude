# Scope

## v1.0.0の対象

- 検索意図と記事内容の差
- 読者の潜在的不安
- 情報源の透明性
- SERP固有エンティティの保護
- 内部リンクの意味的評価
- 読者の判断支援
- 日付・価格・仕様・安全情報の更新性

## 対象外

- Review Confidenceのスコアリング
- Comparison Confidence
- Intent Similarity Score
- Cannibalization Analyzer
- Freshness Monitorの自動化
- 製品固有のJSON Contract
- ArticleID、SiteID等の識別情報管理
