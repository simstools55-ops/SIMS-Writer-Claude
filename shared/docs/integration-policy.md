# Integration Policy

## 正本とスナップショット

このリポジトリを共通知識の正本とする。各製品には、特定バージョンの内容を検証済みスナップショットとして同梱する。

## 更新手順

1. 共通知識を変更する
2. 共通知識のテストを実行する
3. VERSIONとCHANGELOGを更新する
4. リリースタグを作成する
5. 対象製品へスナップショットを取り込む
6. 製品固有の回帰テストを実行する
7. 製品をリリースする

## 禁止事項

- WriterとArticle Creatorで別々に同じ共通知識を編集する
- 未リリースの共通知識を製品へ自動追従させる
- 共通知識だけの変更で製品回帰テストを省略する
