# Evidence-Strength Wording

## Principle
表現の強さをEvidenceの強さに合わせる。

| Evidence | 推奨表現 |
|---|---|
| 公式一次情報 | 公式に発表・掲載されている |
| 正規取扱情報 | 正規取扱店として案内されている |
| モール掲載 | 取り扱い・出品が確認できる |
| 報道・専門媒体 | 報道・資料では〜とされている |
| 限定的資料 | 今回確認した資料では〜 |
| 推定 | 可能性がある／考えられる |

「公式」「最安」「唯一」「必ず」「効果がある」などの強い語は、同等に強い根拠がある場合だけ使用する。
