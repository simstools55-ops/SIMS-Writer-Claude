# Publication QA Validation

## Required Checks

1. 公開品質
2. 検索意図と主要回答
3. Winner Query Preservation
4. 既存資産の保全
5. 数値・論理・事実整合性
6. 内部リンクの意味と実装状態
7. 出力Contract
8. Validation自己整合性
9. YMYL・安全性

## Blocking Examples

- 明白な数値矛盾
- 本文とFAQの主要事実矛盾
- 体験・専門性の捏造
- 未確認リンクを実装済みとする表示
- 必須Contract欠落
- 重大欠陥をValidationがPASSと判定
