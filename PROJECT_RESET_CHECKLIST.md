# Claude Project Reset Checklist

1. Open the SIMS Writer Claude Project.
2. Open Project settings and replace the entire Project Instructions text with `PROJECT_INSTRUCTIONS_TO_PASTE.md`.
3. Delete every existing Project Knowledge file.
4. Upload only the files from this `claude/` package.
5. Start a new chat inside the Writer Project.
6. Send the same existing-article improvement request again.

Important: Replacing Knowledge files does not automatically replace the Project Instructions text stored in Claude Project settings.
