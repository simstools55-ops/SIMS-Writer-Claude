# Rewrite Level and Scope Specification v1.0

## 1. Rewrite Level — 深さ

### L0：変更なし

- 問題なし
- monitorのみ

### L1：軽微修正

- 誤字
- 表記
- 語尾
- 小さな説明補正
- 強すぎる表現の緩和

目安：0〜5%

### L2：局所修正

- タイトル
- 導入
- FAQ
- 1〜2段落の修正
- 小規模な見出し修正

目安：5〜15%

### L3：段落追加・局所本文改善

- 必須論点の追加
- H2内の複数段落修正
- 具体例や計算例の追加
- 不足FAQの追加

目安：15〜30%

### L4：構成再設計

- H2順序変更
- 複数セクション再構築
- 検索意図へ合わせた本文再配置
- 重複削除

目安：30〜60%

### L5：全面書き換え候補

- 中心結論の再設計
- 記事全体の構造変更
- 大半の本文差し替え

目安：60〜100%

## 2. Rewrite Scope — 広さ

### S0：変更なし

### S1：単一要素

- SEOタイトルのみ
- FAQ1件のみ
- 説明1段落のみ

### S2：冒頭・補助要素

- タイトル＋導入
- 導入＋FAQ
- メタ情報＋冒頭

### S3：H2単位

- 特定H2と配下本文
- H2＋関連FAQ
- 局所内部リンク

### S4：本文全体

- 複数H2
- 本文構造全体
- 導入・本文・まとめ

### S5：記事全体

- タイトル
- 導入
- 全見出し
- 本文
- FAQ
- まとめ
- 内部リンク

## 3. 組み合わせ例

- L1/S1：表記修正
- L2/S2：タイトルと導入の改善
- L3/S3：特定H2へ段落追加
- L4/S4：本文構成の再設計
- L5/S5：全面書き換え候補

## 4. 不自然な組み合わせ

- L5/S1
- L1/S5
- Preservation 90以上でL4/S4
- LOW_SAMPLEでL5/S5
- PROTECT_WINNERでL4以上

該当時はSemantic Validatorへ警告を渡す。

## 5. 外部出力への変換

Rewrite Level/Scopeは内部値。

外部では次へ反映する。

- `improvement_type`
- `changes`
- `next_action`
- `summary`
- `warnings`
