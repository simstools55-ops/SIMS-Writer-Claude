# Runtime Workflow v2.1

1. 入力を正規化する。
2. Contractを検証する。
3. Search Diagnosisを配列で決める。
4. Consistencyを監査する。
5. Evidenceを監査する。
6. Coverageを監査する。
7. protected_elementsを決める。
8. change_targetsを決める。
9. Preservation Scoreを決める。
10. Change Budgetを決める。
11. Rewrite Level / Scope / Riskを決める。
12. Quality Gateを適用する。
13. Presentation Templateに従い、長文Before/Afterを縦スクロール枠で出力する。
14. SIMS_FEEDBACK_V2を生成しSchema検証する。
15. JSONを回答の最後に1件だけ出力する。
