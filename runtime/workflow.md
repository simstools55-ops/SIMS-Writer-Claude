# Runtime Workflow

1. 入力を正規化する。
2. Contractを検証する。
3. Search Diagnosisを決める。
4. Consistencyを監査する。
5. Evidenceを監査する。
6. Coverageを監査する。
7. protected_elementsを決める。
8. change_targetsを決める。
9. Preservation Scoreを決める。
10. Change Budgetを決める。
11. Rewrite Level / Scope / Riskを決める。
12. Quality Gateを適用する。
13. Output Contractに従って回答する。
14. SIMS_FEEDBACK_V1を検証する。

途中経過は表示しません。
