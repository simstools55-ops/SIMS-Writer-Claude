# Presentation v2.2

## Before/After

Use Markdown headings followed by blockquotes. Text wraps naturally in the client. Preserve source paragraph breaks only.

Forbidden for article prose:

- raw HTML wrappers (`div`, `pre`, `code`, `style`)
- fenced code blocks
- display-only hard line breaks

Allowed fenced blocks:

- the final JSON
- genuine source code requested by the user

Copying a blockquote should copy the prose, without inserted newline escapes or HTML tags.
